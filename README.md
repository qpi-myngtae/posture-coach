# Posture Coach

노트북 웹캠으로 목표 자세를 저장하고, 그 자세에서 벗어나면 알림을 주는 피지컬 컴퓨팅 대회용 웹 앱입니다.

## VS Code에서 실행

1. VS Code에서 이 폴더를 엽니다.
2. 터미널을 엽니다.
3. 아래 명령을 실행합니다.

```powershell
python server.py
```

4. 브라우저가 열리면 카메라 권한을 허용합니다.

## 사용법

- `카메라 시작`: 웹캠과 AI 자세 인식을 시작합니다.
- `목표 자세 저장`: 바른 자세를 3초 동안 저장합니다.
- `s` 키: 목표 자세 저장 단축키입니다.
- `민감도`: 낮을수록 자세 변화에 예민하게 반응합니다.
- `알림 지연`: 자세가 벗어난 뒤 몇 초 후 알림을 낼지 정합니다.

## 구현 방식

- MediaPipe Pose Landmarker Web을 사용해 웹캠 영상에서 신체 랜드마크를 추출합니다.
- 코, 어깨, 엉덩이 좌표로 머리 위치, 어깨 기울기, 상체 기울기를 계산합니다.
- 처음 저장한 목표 자세와 현재 자세의 차이가 기준값을 넘으면 주의 상태로 표시합니다.
- 기준값을 넘은 상태가 설정한 시간 이상 지속되면 알림음을 재생합니다.
